# Vision-Based Object Detection for Robotic Assembly

This project focuses on developing a YOLO-based object detection system as part of a larger vision-driven robotic assembly pipeline using a KUKA KR6 R900 industrial robot.

The system is designed to detect objects on a conveyor and enable automated pick-and-place operations using a soft robotic gripper.

---

## 🚀 Overview

This project is a key module in a **vision-based automated assembly system**, where a camera is used to detect objects in real time and assist a robotic arm in performing precise operations.

The detection system is trained on a custom dataset and integrated into a pipeline for industrial automation tasks.

---

## 🧠 Key Objectives

- Detect specific objects on a conveyor using YOLO based custom model.
- Train a custom dataset for object recognition.
- Enable real-time detection for robotic applications.
- Developing of the soft gripper using Eco-Flex 00-50
- Support automated pick-and-place using a soft gripper
